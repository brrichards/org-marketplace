# Default Profile

Baseline profile with no plugins enabled. Safe permissions only.

## Available Profiles

Switch profiles with:

```bash
npx tsx scripts/swap-profile.ts swap <profile-name> [target-dir]
```

| Profile | Description |
|---------|-------------|
| `default` | No plugins, safe defaults |
| `example-full` | Enables the example-plugin (commands, agents, skills, hooks, MCP) |
